# Chaos Theory with the Logistic Map

This project explores chaotic behavior in nonlinear dynamical systems using the **logistic map equation**:

xₙ₊₁ = r xₙ (1 − xₙ)

Despite its simple form, the logistic map produces a wide range of behaviors including stable equilibria, periodic oscillations, and chaotic dynamics.

By varying the parameter **r**, the system transitions from predictable behavior to complete chaos.

---

## Key Concepts

This project demonstrates several important mathematical and engineering ideas:

- Nonlinear dynamical systems
- Chaos theory
- Sensitivity to initial conditions
- Period-doubling bifurcations
- Computational modeling with Python
